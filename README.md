# PCO Service Planner

A planning tool for Peace Church Online worship services.

Tracks song rotation, pad suggestions, outfit history and prayer focus for weekly services. Suggests the least recently used songs (openers and closers separately), pads, outfits and prayer topic so there is one less thing to think about each week.

## Files
- `index.html` — the full app (single file, no build process needed)
- `fonts/` — SSNickson One font file

## Features
- Weekly service planner with auto-suggestions
- Song library split into openers and closers
- Pad rotation tracking
- Outfit rotation with colour swatches
- Prayer focus rotation (5-week cycle)
- Full service history back to August 2025

## Hosting
Hosted on GitHub Pages. Any commit to main auto-deploys.

## Updating
1. Make changes in the Claude chat
2. Download the new `index.html`
3. Upload to this repo to replace the existing file
4. GitHub Pages auto-deploys within a minute or two
